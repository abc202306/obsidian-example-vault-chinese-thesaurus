## 配置

> 修改配置变量值以指定文件夹路径

| 配置变量名       | 配置变量值       | 配置变量位置     |
| ----------- | ----------- | ---------- |
| 汉语主题词表数据根目录 | data/汉语主题词表 | 文件夹路径/绝对路径 |
| 主题词子目录      | 主题词         | 文件夹路径/相对路径 |
| 主题词分类表子目录   | 主题词分类       | 文件夹路径/相对路径 |
| 主题词组配分类表子目录 | 主题词组配分类     | 文件夹路径/相对路径 |
